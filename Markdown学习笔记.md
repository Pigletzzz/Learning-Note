## 段落格式

### 段落

* 文本末尾加两个空格换行

* 使用*或_分别在文本两端

`*斜体文本*`或`_斜体文本_`

`**粗体文本**`或`__粗体文本__`

`***斜粗体文本***`或`___斜粗体文本___`

## 列表

### 无序列表

* 使用*、+、-作为列表标记并加一个空格
* 按空格取消列表

`* 标题`或`- 标题`或`+ 标题`

* 第一项
* 第二项
* 第三项

### 有序列表

* 使用数字后加一个.

```
1. 第一项

2. 第二项

3. 第三项
```

	1. 第一项
 	2. 第二项
 	3. 第三项

### 列表嵌套

* 在子列表选项前面加4个空格

```
1. 第一项：
    - 第一项嵌套的第一个元素
    - 第一项嵌套的第二个元素
2. 第二项：
    - 第二项嵌套的第一个元素
    - 第二项嵌套的第二个元素
```

1. 第一项

      - 第一项嵌套的第一个元素
      - 第一项嵌套的第二个元素

2. 第二项

   * 第二项嵌套的第一个元素

       * 第二项嵌套的第二个元素

## 表格

### 制表

* 使用`|`分割不用单元格，用`-`分割表头和其他行

````
|表头|表头|
|---|---|
|单元格|单元格|
|单元格|单元格|
````

| 表头   | 表头   |
| ------ | ------ |
| 单元格 | 单元格 |
| 单元格 | 单元格 |

### 表格格式

>`-:`设置内容和标题左对齐
>
>`:-`设置内容和标题右对齐
>
>`:-:`设置内容和标题居中

````
| 左对齐 | 右对齐 | 居中对齐 |
| :-----| ----: | :----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
````

| 左对齐 | 右对齐 | 居中对齐 |
| :----- | -----: | :------: |
| 单元格 | 单元格 |  单元格  |
| 单元格 | 单元格 |  单元格  |
